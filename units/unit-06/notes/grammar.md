# Unit 06 — Grammar concepts (فعلِ حال / the present-tense verb)

Unit 6 ("چھٹا یونٹ: فعلِ حال") is the **first verb unit**. It teaches the **مضارع
(present/imperfect verb)** — "he does / is doing" — and how it changes for person, number
and gender, how it is negated, and the common Quranic expressions built on it. With these
forms plus the vocabulary of Units 1–5, the book says 1100+ Quranic sentences become
readable. (22 lessons.)

## The big idea — مضارع (present/imperfect verb)
The مضارع names an action in the present/future: "he does / is doing". It is recognised by a
**prefix letter** that shows the doer, and it usually ends in **ـُ**. The four prefix letters
are **أ ن ي ت** (remembered as "أَنَيْتُ"):

- **يَـ** = "he" — يَعْبُدُ (he worships), يَعْلَمُ، يَسْمَعُ، يُؤْمِنُ، يَنْزِلُ.
- **تَـ** = "she" (3rd fem.) **and** "you" (2nd masc. sing.) — تَعْبُدُ (she/you worship).
- **أَـ** = "I" — أَعْبُدُ (I worship).
- **نَـ** = "we" — نَعْبُدُ (we worship).

## سبق 1 — يَـ (he) → تَـ (she)
**Rule:** the مضارع begins with **يَـ** for a masculine doer ("he does"); change the يَـ to
**تَـ** for a feminine doer ("she does"). يَعْبُدُ ("he worships") → تَعْبُدُ ("she worships");
يَسْجُدُ → تَسْجُدُ. (تَـ also serves for "you", masc. sing.)

## Person & number
The same stem takes different prefixes/endings for each doer:
- **I / we:** أَعْبُدُ / نَعْبُدُ.
- **you (m.s.) / she:** تَعْبُدُ.
- **they (masc. pl.):** add **ـُونَ** → يَعْبُدُونَ، يَعْلَمُونَ، يَعْمَلُونَ، يَفْعَلُونَ.
- **you all:** **تَـ … ـُونَ** → تَعْمَلُونَ، تَعْلَمُونَ، تَعْقِلُونَ.

## Negation
- **لَا + مضارع** = present/future negative: لَا يَعْلَمُونَ ("they do not know"),
  لَا يُؤْمِنُونَ، لَا يَشْعُرُونَ.
- **لَمْ + مضارع** = **past** negative ("did not …"); it puts the verb into **jazm** (a
  sukūn / shortened ending): لَمْ يَلِدْ وَلَمْ يُولَدْ، لَمْ يَجْعَلْ، لَمْ يَكُنْ.
- **أَلَمْ …؟** = "have you not …? / did … not …?": ﴿أَلَمْ تَرَ﴾ ("have you not seen?"),
  ﴿أَلَمْ تَعْلَمْ﴾ ("do you not know?"), ﴿أَلَمْ نَشْرَحْ﴾ ("did We not expand?").

## Common Quranic expressions on the مضارع
- **أَفَلَا تَعْقِلُونَ / أَفَلَا تَذَكَّرُونَ / أَفَلَا تَتَفَكَّرُونَ** = "do you not then
  reason / take heed / reflect?" (rhetorical rebuke).
- **لَعَلَّكُمْ تَعْقِلُونَ / تَتَّقُونَ / تَشْكُرُونَ** = "so that you may reason / be mindful
  / be grateful" (purpose).
- **مَا تَعْمَلُونَ** = "what you do"; **بِمَا تَعْمَلُونَ** = "of what you do" — signature:
  ﴿وَاللهُ بِمَا تَعْمَلُونَ بَصِيرٌ / عَلِيمٌ / خَبِيرٌ﴾.
- **يَعْلَمُ مَا …** = "He knows what …": ﴿يَعْلَمُ مَا تُسِرُّونَ وَمَا تُعْلِنُونَ﴾.
