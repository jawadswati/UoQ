# Unit 02 — Grammar concepts (13 lessons)

Unit 02 is grammar-driven. Each lesson (سبق) introduces **one** high-frequency Quranic
particle/operator and drills it on already-known Unit 01 vocabulary, then on real āyāt.
The vocabulary itself is reused from Unit 01, so this file is the core record for the unit;
all Quranic āyāt are collected in `ayaat.csv`.

Pages run right→left across spreads: slide1-left = p52, slide2-right = p53, slide2-left = p54, …

## Quick reference

| # | Particle | Meaning (Urdu / English) | Effect on next word |
|---|----------|--------------------------|---------------------|
| 1 | إِنَّ | بے شک / یقیناً — indeed, certainly | following noun منصوب (zabar) |
| 2 | لِـ | کے لیے — for / belonging to | kasra; with ال → لِلْ |
| 3 | فِي | میں — in | kasra |
| 4 | عَلَى | پر — on / upon | kasra |
| 5 | مِنْ | سے — from | kasra |
| 6 | إِلَى | کی طرف / تک — to / towards | kasra |
| 7 | بِـ | سے / پر / کو / کے بدلے — with, by, in, in exchange for | kasra |
| 8 | لَا | (قطعاً) کوئی … نہیں — absolute negation (لاءِ نفیِ جنس) | fatha, no tanwīn |
| 9 | إِلَّا | مگر / سوائے — except, but | (also مَا … = negation) |
| 10 | وَ | … کی قسم — oath "by" (واوِ قسم) | kasra |
| 11 | أَنَّ | کہ — that | following noun منصوب (zabar) |
| 12 | كَانَ | تھا — was | predicate (خبر) becomes منصوب |
| 13 | يَا | اے — O (vocative) | name; for ال words → يَا أَيُّهَا |

## سبق 1 — الْكِتَابُ ← إِنَّ الْكِتَابَ (إِنَّ)
**Rule:** إِنَّ = "بے شک / یقیناً". Comes at sentence start; the following noun takes zabar (منصوب).
| | |
|--|--|
| الْإِنْسَانُ → إِنَّ الْإِنْسَانَ | بے شک انسان |
| الشِّرْكُ → إِنَّ الشِّرْكَ | بے شک شرک |
| الْمَوْتُ → إِنَّ الْمَوْتَ | بے شک موت |

Variants: **فَإِنَّ** = "تو بیشک"; with emphatic **لَـ** in the predicate: ﴿إِنَّ هٰذَا لَسَاحِرٌ﴾.
Drill words: إِنَّ الدِّينَ، إِنَّ الْهُدَى، إِنَّ الْعَاقِبَةَ، إِنَّ الْعِزَّةَ، إِنَّ جَهَنَّمَ،
إِنَّ الظَّنَّ (گمان)، إِنَّ إِبْرَاهِيمَ، إِنَّ فِرْعَوْنَ، إِنَّ قَارُونَ، إِنَّ يَأْجُوجَ.

## سبق 2 — الْكِتَابُ ← لِلْكِتَابِ (لِـ)
**Rule:** لِـ = "کے لیے". Attaches with kasra; before ال becomes لِلْ.
| | |
|--|--|
| الْإِسْلَامُ → لِلْإِسْلَامِ | اسلام کے لیے |
| الْإِنْسَانُ → لِلْإِنْسَانِ | انسان کے لیے |
| الزَّكَاةُ → لِلزَّكَاةِ | زکوٰۃ کے لیے |

Indefinite: لِـ + word = "کسی/ایک کے لیے" (لِنَبِيٍّ، لِقَوْمٍ، لِرَجُلٍ).
Sentence forms: **لله** (الْحَمْدُ لِلهِ)، **وَلِله** (oath/emphasis, وَلِلهِ الْعِزَّةُ)، **فَلِله** (تو اللہ کے لیے).

## سبق 3 — الْكِتَابُ ← فِي الْكِتَابِ (فِي)
**Rule:** فِي = "میں". Next word takes kasra.
| | |
|--|--|
| الْأَرْضُ → فِي الْأَرْضِ | زمین میں |
| الْقُرْآنُ → فِي الْقُرْآنِ | قرآن میں |
| الْكِتَابُ → فِي الْكِتَابِ | کتاب میں |

Indefinite: فِي شَكٍّ، فِي رَحْمَةٍ، فِي ضَلَالٍ، فِي قَرْيَةٍ. Emphatic **لَفِي**: ﴿إِنَّ الْإِنْسَانَ لَفِي خُسْرٍ﴾.
Signature āyah pattern: ﴿إِنَّ فِي ذٰلِكَ لَآيَةً / لَعِبْرَةً / لَذِكْرَى﴾.

## سبق 4 — الْكِتَابُ ← عَلَى الْكِتَابِ (عَلَى)
**Rule:** عَلَى = "پر". Next word takes kasra.
| | |
|--|--|
| الْأَرْضُ → عَلَى الْأَرْضِ | زمین پر |
| الْإِيمَانُ → عَلَى الْإِيمَانِ | ایمان پر |
| الْغَيْبُ → عَلَى الْغَيْبِ | غیب پر |

Indefinite: عَلَى هُدًى، عَلَى عِلْمٍ، عَلَى شَيْءٍ. Emphatic **لَعَلَى**: ﴿وَإِنَّكَ لَعَلَى خُلُقٍ عَظِيمٍ﴾.
Signature: ﴿سَلَامٌ عَلَى نُوحٍ / إِبْرَاهِيمَ / مُوسَى وَهَارُونَ﴾.

## سبق 5 — الْكِتَابُ ← مِنَ الْكِتَابِ (مِنْ)
**Rule:** مِنْ = "سے". Next word takes kasra.
| | |
|--|--|
| الْأَرْضُ → مِنَ الْأَرْضِ | زمین سے |
| الْقُرْآنُ → مِنَ الْقُرْآنِ | قرآن سے |
| الْكِتَابُ → مِنَ الْكِتَابِ | کتاب سے |

Indefinite: مِنْ نَارٍ، مِنْ طِينٍ (مٹی)، مِنْ مَاءٍ، مِنْ خَوْفٍ.
Comparative use: ﴿وَالْفِتْنَةُ أَكْبَرُ / أَشَدُّ مِنَ الْقَتْلِ﴾.

## سبق 6 — الْكِتَابُ ← إِلَى الْكِتَابِ (إِلَى)
**Rule:** إِلَى = "کی طرف / تک". Next word takes kasra.
| | |
|--|--|
| الْإِسْلَامُ → إِلَى الْإِسْلَامِ | اسلام کی طرف |
| الْإِنْسَانُ → إِلَى الْإِنْسَانِ | انسان کی طرف |
| الرَّسُولُ → إِلَى الرَّسُولِ | رسول کی طرف |

Indefinite: إِلَى قَوْمٍ، إِلَى أَجَلٍ، إِلَى صِرَاطٍ.
Signature pairing: ﴿مِنَ الظُّلُمَاتِ إِلَى النُّورِ﴾.

## سبق 7 — الْكِتَابُ ← بِالْكِتَابِ (بِـ)
**Rule:** بِـ has several senses by context — "سے / پر / کو / یا / و", and **"کے بدلے" (in exchange for)**.
Next word takes kasra.
| | |
|--|--|
| الْإِيمَانُ → بِالْإِيمَانِ | ایمان پر / سے / کو |
| الْقُرْآنُ → بِالْقُرْآنِ | قرآن پر / سے / کو |
| الْقَلَمُ → بِالْقَلَمِ | قلم سے |

"in exchange for": ﴿النَّفْسَ بِالنَّفْسِ وَالْعَيْنَ بِالْعَيْنِ … وَالسِّنَّ بِالسِّنِّ﴾؛ ﴿الْحُرُّ بِالْحُرِّ وَالْعَبْدُ بِالْعَبْدِ وَالْأُنْثَى بِالْأُنْثَى﴾.

## سبق 8 — كِتَابٌ ← لَا كِتَابَ (لَا — absolute negation)
**Rule:** لَا (لاءِ نفیِ جنس) negates the whole genus = "قطعاً کوئی … نہیں". Next word takes fatha, no tanwīn.
| | |
|--|--|
| خَيْرٌ → لَا خَيْرَ | قطعاً کوئی بھلائی نہیں |
| عِلْمٌ → لَا عِلْمَ | بالکل کوئی علم نہیں |
| قُوَّةٌ → لَا قُوَّةَ | ہرگز کوئی قوت نہیں |

Variant **فَلَا** = "تو قطعاً کوئی … نہیں" (فَلَا جُنَاحَ، فَلَا نَاصِرَ).
Signature: ﴿لَا رَيْبَ فِيهِ﴾، ﴿لَا إِكْرَاهَ فِي الدِّينِ﴾، ﴿لَا إِلٰهَ إِلَّا اللهُ﴾.

## سبق 9 — الْكِتَابُ ← إِلَّا الْكِتَابَ (إِلَّا) + مَا
**Rule:** إِلَّا = "مگر / سوائے". Often paired with a negation. مَا at sentence start = "نہیں".
| | |
|--|--|
| اللهُ → إِلَّا اللهَ | مگر اللہ / سوائے اللہ کے |
| رَسُولٌ → إِلَّا رَسُولٌ | مگر رسول |
| إِبْلِيسُ → إِلَّا إِبْلِيسَ | مگر ابلیس |

With prepositions: إِلَّا بِالْحَقِّ، إِلَّا لِلهِ، إِلَّا فِي ضَلَالٍ.
مَا … إِلَّا: ﴿مَا هٰذَا إِلَّا بَشَرٌ﴾، ﴿وَمَا مُحَمَّدٌ إِلَّا رَسُولٌ﴾.

## سبق 10 — الْكِتَابُ ← وَالْكِتَابِ (وَ — oath)
**Rule:** the **oath wāw** (واوِ قسم) before a noun = "… کی قسم" (by …). Next word takes kasra.
(Distinct from the "and" wāw of Unit 01 slide 12.)
| | |
|--|--|
| اللهُ → وَاللهِ | اللہ کی قسم |
| الْقُرْآنُ → وَالْقُرْآنِ | قرآن کی قسم |
| الْقَلَمُ → وَالْقَلَمِ | قلم کی قسم |

Signature: ﴿وَالْعَصْرِ، إِنَّ الْإِنْسَانَ لَفِي خُسْرٍ﴾؛ ﴿يس، وَالْقُرْآنِ الْحَكِيمِ﴾.

## سبق 11 — الْكِتَابُ ← أَنَّ الْكِتَابَ (أَنَّ)
**Rule:** أَنَّ = "کہ". Following noun takes zabar (منصوب).
| | |
|--|--|
| الْمَاءُ → أَنَّ الْمَاءَ | کہ پانی |
| الْأَرْضُ → أَنَّ الْأَرْضَ | کہ زمین |
| الْعَذَابُ → أَنَّ الْعَذَابَ | کہ عذاب |

Signature: ﴿وَأَنَّ الْمَسَاجِدَ لِلهِ﴾، ﴿وَأَنَّ السَّاعَةَ آتِيَةٌ﴾، ﴿أَنَّ الْقُوَّةَ لِلهِ جَمِيعًا﴾.

## سبق 12 — رَسُولٌ ← كَانَ رَسُولًا (كَانَ)
**Rule:** كَانَ = "تھا". The predicate (خبر) becomes منصوب; an indefinite predicate takes ـًا (alif).
| | |
|--|--|
| رَسُولٌ → كَانَ رَسُولًا | وہ رسول تھا |
| مَرِيضٌ → كَانَ مَرِيضًا | وہ بیمار تھا |
| مُؤْمِنٌ → كَانَ مُؤْمِنًا | وہ مومن تھا |

Drill: كَانَ حَيًّا (زندہ)، كَانَ حَنِيفًا (یکسو)، كَانَ يَسِيرًا (آسان)، كَانَ ظَلُومًا، كَانَ مَسْطُورًا (لکھا ہوا).
Signature: ﴿وَكَانَ اللهُ غَفُورًا رَحِيمًا﴾، ﴿وَكَانَ الْإِنْسَانُ كَفُورًا / عَجُولًا / قَتُورًا﴾.

## سبق 13 — آدَم ← يَا آدَم (يَا — vocative)
**Rule:** يَا = "اے" (O), for calling. Before an ال-word use **يَا أَيُّهَا**.
| | |
|--|--|
| آدَمُ → يَا آدَمُ | اے آدم |
| نُوحٌ → يَا نُوحُ | اے نوح |
| النَّبِيُّ → يَا أَيُّهَا النَّبِيُّ | اے نبی |

يَا أَيُّهَا examples: يَا أَيُّهَا النَّاسُ، يَا أَيُّهَا الْإِنْسَانُ، يَا أَيُّهَا الْمُزَّمِّلُ (چادر لپیٹنے والے)،
يَا أَيُّهَا الْمُدَّثِّرُ (چادر اوڑھنے والے)، يَا أَيُّهَا السَّاحِرُ، أَيُّهَا الْكَافِرُونَ.
