# Unit 02 — Exercises (مشق)

Each lesson ends with a مشق block. Because Unit 02 teaches particles, the exercises are
sentence-level (more āyah work than Unit 01). The recurring types:

1. **درج ذیل آیات میں صرف اس حصے کو نمایاں کریں جس میں (particle) ہے** — In each āyah,
   highlight the part containing the lesson's particle and the word it attaches to (a worked
   first example is given). → these āyāt are all in `ayaat.csv` (topic = the particle).
2. **(particle) کے بعد آنے والے قطعوں کا زبانی ترجمہ کریں** — Orally translate each
   "particle + word" chunk (the drill word-lists; these reuse Unit 01 vocabulary).
3. **مکمل/نفی طور پر بنائیں** (lesson 8, لَا) — Form the absolute-negation of given words.
4. **(کن چیزوں کی) قسم کھائی گئی ہے؟** (lesson 10, oath wāw) — Identify what is sworn by.

## Worked example — Lesson 1 (إِنَّ), مشق سوال 1

> درج ذیل الفاظ/جملوں کا ترجمہ کریں: إِنَّ الدِّينَ، إِنَّ الشِّرْكَ، إِنَّ الصَّلَاةَ،
> إِنَّ هٰذَا، إِنَّ الْمَوْتَ، إِنَّ الْهُدَى، إِنَّ الْعِزَّةَ، إِنَّ الْخِزْيَ،
> إِنَّ جَهَنَّمَ، إِنَّ الْبَاطِلَ، إِنَّ الْقَوْمَ، إِنَّ الْفَضْلَ، إِنَّ النَّفْسَ،
> إِنَّ الْعَهْدَ، إِنَّ الْأَمْرَ.
