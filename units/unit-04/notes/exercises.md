# Unit 04 — Exercises (مشق)

Each pronoun lesson ends with a مشق. Because Unit 4 teaches pronouns through a fixed
four-step cycle (detached → suffix on noun → suffix after particle → suffix on particle),
the exercises drill the same cycle. Recurring types:

1. **قرآنی قطعوں کا ترجمہ** — Translate the Quranic chunks containing the lesson's pronoun,
   e.g. لَهُ الْمُلْكُ، لَهُ الْهُدَى، فِيهِ هُدًى، إِنَّهُ هُوَ الْغَفُورُ الرَّحِيمُ.
2. **ضمیر کا مطلب بتائیں** — State what each detached pronoun / suffix means
   (هُوَ، أَنْتَ، هِيَ، أَنَا ؛ ـهُ، ـكَ، ـهَا، ـي).
3. **اسم پر ضمیر لگائیں** — Attach the suffix to a base noun (بَيْت → بَيْتُهُ / بَيْتُكَ /
   بَيْتُهَا / بَيْتِي).
4. **حرف کے ساتھ ضمیر** — Join the suffix to a particle (لِـ → لَهُ / لَكَ / لَهَا / لِي ؛
   مِنْ → مِنْهُ / مِنْكَ / مِنْهَا / مِنِّي).
5. **نمایاں کریں** — Highlight the pronoun (detached or suffix) in each Quranic phrase.

The full Quranic phrases used are in `ayaat.csv`; the pronoun forms and base nouns are in
`vocabulary.csv` (sections `detached-pronouns`, `attached-pronouns`, `pronoun-on-particle`,
`noun-bases`); the rules and examples are in `grammar.md`.
