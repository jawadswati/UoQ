# ʿIlm-ul-Qurʾān — Teaching Material / علم القرآن

Study material for learning the **language of the Qurʾān** the easy way — building first on
vocabulary that is already familiar from Urdu, then gradually adding simple grammar.
Based on the **معلم القرآن (Muallim ul Quran)** course.

> یہ مواد قرآن کی زبان آسان انداز میں سیکھنے کے لیے ہے — پہلے وہ الفاظ جو اردو میں پہلے سے
> مانوس ہیں، پھر بتدریج آسان قواعد۔

## How to use this repository / استعمال کا طریقہ

Everything is organised **by unit**. Open a unit folder and you will find up to three parts:

| Folder | What's inside |
|--------|---------------|
| **`slides/`** | The lesson slides (images) — read these first. |
| **`notes/`** | Word lists, grammar, phrases and Quranic verses transcribed from the slides (for quick reference, flashcards and revision). |
| **`practice/`** | Bilingual (English + Urdu) question bank with answers — for self-testing and exams. |

## What's in the notes / نوٹس میں کیا ہے

- **`vocabulary.csv` / `vocabulary.md`** — every word with its Urdu meaning (great for flashcards).
- **`grammar.md`** — each grammar lesson with the rule, examples, and Quranic phrases.
- **`ayaat.csv`** — the Quranic verses/phrases used in the unit's exercises.
- **`phrases.csv`** (Unit 3) — fixed two-word expressions (موصوف+صفت, اِضافت) with meanings.
- **`exercises.md`** — the exercise types used in the book's مشق.

The `.csv` files open in Excel/Google Sheets so you can sort, filter, and make your own quizzes.

## A note on accuracy

The notes were transcribed from the slide images. A few entries are marked with a
`confidence` of `med`/`low` (or `[?]`) — please double-check those against the slide before
relying on them in an exam.
